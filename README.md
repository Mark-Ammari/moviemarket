## MovieMarket Project ##
    MERN Stack Project that utilizes themoviedb Api. User's can roam around our website to read about tv shows and movies. You can search, signup, and add or remove assets from your favorites.
___

### Prominent Libraries/Tools Used ###
    * React
    * React-Redux, Redux, & Redux-Thunk
    * React-Router
    * Material UI 
    * MongoDB
    * NodeJS
    * ExpressJS
    * Express-Session
    * BcryptJS
    * Axios
